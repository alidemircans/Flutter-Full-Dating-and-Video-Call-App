Change logo icon & screen logo #back to top

Change logo icon & screen logo Replace icon.png file inside /asset/icon/ with your logo and keep the same file name i.e. “icon.png“. Save and run “flutter pub run flutter_launcher_icons:main” on the terminal (project_root). For screen logo replace hookup4u-Logo-BP.png file inside path_to_project_root>/asset/ with your logo and keep the same file name i.e. “hookup4u-Logo-BP.png“. Change splash Screen logo For splash Screen logo replace hookup4ulogobp.png file inside /android/app/src/main/res/drawable/hookup4ulogobp.png. with your logo and keep the same file name i.e. “hookup4ulogobp.png“. Change Colors Go to /lib/util/color.dart And replace the primary and secondary colors of your choice.
